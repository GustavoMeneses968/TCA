# Misnemom RPG

## Descrição do nosso projeto
Nosso projeto é um RPG 2D, onde existem 4 inimigos e 1 Boss. Os 4 inimigos tem IA para andarem sozinhos,
ou seja eles tem uma rota predefinida e se você estiver perto o suficiente ele começara a te perserguir.
